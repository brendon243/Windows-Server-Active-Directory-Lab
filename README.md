Designed and implemented a Windows Server Active Directory lab, including domain controller promotion, DNS configuration, and domain-joined clients. Practiced core IT support workflows such as user provisioning, password resets, and access management.

Overview
This project demonstrates the deployment of a Windows Server Active Directory environment,
including domain controller configuration, DNS setup, user and group management,
and domain-joined client machines.

Environment
- Windows Server 2022 (Domain Controller)
- Windows 10 Client
- Active Directory Domain Services (AD DS)
- DNS
- Virtualized environment (Azure / VMware / VirtualBox)

Project Steps
1. Deployed a Windows Server VM
2. Installed and configured Active Directory Domain Services
3. Promoted server to Domain Controller
4. Created domain users, groups, and OUs
5. Joined client machine to domain
6. Performed account administration (password resets, unlocks)

Skills Demonstrated
- Windows Server Administration
- Active Directory
- DNS Configuration
- User & Group Management
- IT Support Troubleshooting
